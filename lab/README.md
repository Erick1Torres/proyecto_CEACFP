# Laboratorio — pfSense · Suricata · Auditoría

Este directorio agrupa la **implementación documentada** del enunciato (firewall perimetral, IDS/IPS y línea de auditoría/refuerzo) **fuera del navegador**, lista para reproducir en VMs.

| Carpeta | Contenido |
|---------|-----------|
| [`pfsense/`](pfsense/README.md) | Pasos de despliegue, criterios de reglas, ejemplo comentado de política WAN/LAN |
| [`suricata/`](suricata/README.md) | Suricata en pfSense (recomendado en este proyecto), modos IDS/IPS, pruebas |
| [`auditoria/`](auditoria/README.md) | Alcance de pentest en laboratorio, mitigaciones, vínculo con el panel web |
| [`LIMITACIONES-Y-ALTERNATIVAS.md`](LIMITACIONES-Y-ALTERNATIVAS.md) | Qué no cabe en Git y cómo combinarlo con el visor |

**La web del repositorio** (`index.html`, `attack-store.js`) **no sustituye** a pfSense ni Suricata; complementa la memoria con un **registro trazable** de incidentes simulados o documentados.
